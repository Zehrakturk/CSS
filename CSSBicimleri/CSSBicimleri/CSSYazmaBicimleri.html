Internal CSS

Head tagları içerisinde yazılan biçim düzenlemelerine Internal CSS denir.

Internal CSS de Head bölümünün içerisinde bir "style" etiketi açılır ve içerisinde CSS kodları yazılır.

Inline CSS

Istenilen html tagının CSS özelliğini değiştirmek için kullanılır.

Class ve id inline css ile verilir.

Text in olduğu tagın içerisinde istediğimiz özellik verilir. Yani o tag özelleştirilmiş olur.

External CSS

Bir web sayfasına bağlanan bağımsız bir css dosyasıdır.

Head içerisinde link vererek yazılır.
